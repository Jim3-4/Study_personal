  css 특수성(특이성) ?  
  동일한 요소를 가리키는 css 규칙이 두 개이상 있을 경우 어떤 css 규칙을 요소에 적용할 것인지... 
  점수로 순위가 정해진다.
  [특이성 수준을 나타내는 점수]
  1.  인라인 css 적용 :   1000 점  ( 인라인 스타일 적용 방식이 항상 가장 우선순위가 높다. )
     

  2.  id 속성             :   100 점
     #mydiv{     }
  3. 태그명, 의사요소                          : 1점
  4. 클래스명, 의사클래스, 속성 선택자 :  10점
  5. *               :   0점        
  
  
  예)
   p{}               1점
   p.test           1 + 10 = 11점
   p#demo       1 + 100 = 101점
   #demo         100 점
   .test            10점
   p.test1.test2       1+10+10=21점
   #navbar p#demo     100 + 1 + 100 = 201점
   *                                 0
