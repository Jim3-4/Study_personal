ex06_02.html - HTML DOM 탐색

   1. html 문서의 모든 것은 Node(노드) 이다.
   2. 전체 문서 == 문서 노드
      요소노드(element node)
      속성노드(attribute node)
      텍스트노드(text node)
      주석노드(comment node)
   3. 모든 노드들은 관계
     - 부모-자식 관계  
       parent  -  child
       parentNode   childNode
      - 형제 관계
           nextElementSibling 다음 형제 요소
           nextSibling              다음 형제 노드
          previousElementSibling 다음 형제 요소
          previousSibling              다음 형제 노드
     - 자식 노드 
           - 첫 전째 자식 노드 : firstChild
               마지막 자식 노드 : lastChild
               
    4. 노드 들 간의 탐색할 때 자주 사용되는 속성               
       ㄱ. parentNode 속성 : 부모노드
       ㄴ. childNodes 속성 :  자식노드 목록      () nodeList )
       ㄷ. firstChild 속성 
       ㄹ. lastChild 속성 
       ㅁ. nextSibling 속성 
       ㅂ. previousSibling   속성
