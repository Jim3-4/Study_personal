  1. 제이쿼리란?
   - jQuery는 빠르고 작고 기능이 풍부한 "JavaScript 라이브러리""입니다.
   - 여러 브라우저에서 작동하는 사용하기 쉬운 API를 사용하여
      ㄱ.  HTML 문서 탐색 및 조작,
      ㄴ.  이벤트 처리, 
      ㄷ. 애니메이션 및 Ajax
      와 같은 작업을 "훨씬 더 간단"하게 만듭니다.
     - 다목적성과 확장성의 조합으로 jQuery는 수백만 명의 사람들이 JavaScript를 작성하는 방식을 변화시켰습니다.
  
  2. jquery 사용하는 방법
      ㄱ. jquery  다운로드 
          1) 압축 - 대역폭을 절약하고 프로덕션 성능을 향상  jquery-3.6.0.min.js
          2) 비압축 - 개발 또는 디버깅 중에 가장 잘 사용    jquery-3.6.0.js
      ㄴ. CDN( Content delivery network ) 호스팅해서 사용하는 방법
          직접 다운로드 X ( 잇점 )
           전 세계 분포되어 있는 jquery 제공해 주는 서버를 이용.
